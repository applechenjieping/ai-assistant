# AI运营助手 - 产品需求文档 (PRD)

## 1. 产品概述

### 1.1 产品定位
美林湖社区活动智能问答系统，为社区活动提供AI智能FAQ问答服务。

### 1.2 核心价值
- 7×24小时智能问答服务
- 统一入口，多活动并行支持
- 自动识别用户问题归属
- 数据驱动持续优化

---

## 2. 用户故事

### 2.1 社区居民（用户）
| 场景 | 需求 |
|------|------|
| 了解活动 | 快速获取活动信息 |
| 报名咨询 | 解答报名疑问 |
| 问题反馈 | 未解答问题自动收集 |

### 2.2 运营管理员
| 场景 | 需求 |
|------|------|
| 活动管理 | 创建/编辑/上下线活动 |
| FAQ维护 | 批量导入/手动添加FAQ |
| 数据分析 | 查看访问量/热门问题/未解答问题 |
| 每日复盘 | 自动收到日报邮件 |

---

## 3. 功能需求

### 3.1 活动管理
- **创建活动**：名称、描述、时间、地点、热线
- **编辑活动**：修改活动信息
- **删除活动**：下架活动
- **独立链接**：每个活动有分享URL

### 3.2 FAQ知识库
- **批量导入**：Word/PDF/TXT格式解析
- **手动添加**：逐条新增FAQ
- **FAQ字段**：问题、答案、分类、关键词、排序

### 3.3 AI智能问答
- **统一入口**：一个页面支持所有活动
- **自动识别**：AI判断问题属于哪个活动
- **上下文记忆**：多轮对话，理解"那"、"它"等代词
- **未解答收集**：记录FAQ无法回答的问题

### 3.4 数据统计
- 访问量统计
- 热门问题排行TOP10
- 未解答问题列表

### 3.5 每日报送
- **发送时间**：每天09:00（北京时间）
- **发送内容**：访问量、热门问题、未解答问题
- **未解答建议**：给出FAQ补充建议
